EM PROGRESSO - Projeto Greentech

O site GreenTech simplifica a análise de dados de estufas indoor. Unindo Flask WebApp e Dash com uma interface simplificada, permite aos usuários fazer upload de arquivos CSV e visualizar gráficos detalhados. Com foco na acessibilidade, ajuda a entender e otimizar o ambiente das estufas para um melhor crescimento das plantas. É a ferramenta ideal para explorar e interpretar dados de forma eficiente.

EM PROGRESSO - Portfolio Digital

Este projeto consiste em um portfólio digital que destaca minhas habilidades como desenvolvedor, apresentando uma variedade de informações pertinentes. A plataforma inclui seções dedicadas às minhas capacidades, detalhando minha experiência em desenvolvimento e design, bem como os projetos em que estou atualmente envolvido. Além disso, uma página de contato oferece links para minhas contas profissionais, como LinkedIn e GitHub, facilitando a conexão com outros profissionais e potenciais colaboradores. O objetivo principal é fornecer uma visão abrangente das minhas habilidades e experiências, além de facilitar a comunicação e networking dentro da comunidade de desenvolvedores de software.

ESTÁGIOS INICIAIS - Assistente de Vendas ML

O Assistente de Vendas ML é um projeto que visa criar um assistente de vendas de celulares online que utiliza dados do Mercado Livre, uma das maiores plataformas de comércio eletrônico da América Latina, para fornecer aos clientes uma experiência de compra personalizada e conveniente. Combinando tecnologias de processamento de linguagem natural, banco de dados e inteligência artificial, nosso assistente de vendas visa simplificar o processo de compra de celulares, ajudando os clientes a encontrar o dispositivo perfeito para suas necessidades.
