Projeto Greentech

O site GreenTech simplifica a análise de dados de estufas indoor. Unindo Flask WebApp e Dash com uma interface simplificada, permite aos usuários fazer upload de arquivos CSV e visualizar gráficos detalhados. Com foco na acessibilidade, ajuda a entender e otimizar o ambiente das estufas para um melhor crescimento das plantas. É a ferramenta ideal para explorar e interpretar dados de forma eficiente.

Assistente de Vendas ML

O Assistente de Vendas ML é um projeto que visa criar um assistente de vendas de celular online que utiliza dados do Mercado Livre, uma das maiores plataformas de comércio eletrônico da América Latina, para fornecer aos clientes uma experiência de compra personalizada e conveniente. Combinando tecnologias de processamento de linguagem natural, banco de dados e inteligência artificial, nosso assistente de vendas visa simplificar o processo de compra de celulares, ajudando os clientes a encontrar o dispositivo perfeito para suas necessidades.
